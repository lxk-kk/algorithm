题目描述：

```
设计一个算法，找出数组中最小的k个数。以任意顺序返回这k个数均可
```

+ 题解：

  ```
   *  只需要找出前 K 小的数就行了，不需要多余的操作，这么说，严格排序就没有必要了！
   *  可以只排一下 元素 之间的相对位置，找到一个 K 点，使得前 k 个元素就是整个数组最小的 K 个数
   *  那么，就对数组分割，还记得快排是怎么分割数组的嘛！
  ```

  ```java
  /**
   * @author 10652
   */
  public class Solution {
      private int[] arr;
      public int[] smallestK(int[] arr, int k) {
          if(arr==null || arr.length==0 || k<=0){
              return new int[]{};
          }
          this.arr = arr;
          int st = 0;
          int end = arr.length;
          while (true) {
              int tempK = findK(st, end);
              if (tempK < k) {
                  st = tempK + 1;
              } else if (tempK > k) {
                  end = tempK;
              } else {
                  return Arrays.copyOf(arr, k);
              }
          }
      }
  
      private int findK(int st, int end) {
          int num = arr[st];
          int i = st;
          int j = end;
          do {
              i++;
              j--;
              while (i < end && arr[i] <= num) {
                  i++;
              }
              while (j > st && arr[j] >= num) {
                  j--;
              }
              if (i < j) {
                  int temp = arr[i];
                  arr[i] = arr[j];
                  arr[j] = temp;
              }
          } while (i < j);
          int temp = arr[j];
          arr[j] = num;
          arr[st] = temp;
          return j;
      }
  }
  ```
